# TextGO Hub

This repository contains the official website, documentation, and extensions for [**TextGO**](https://github.com/C5H12O5/TextGO).

## About TextGO

TextGO is a cross-platform desktop application designed to help you quickly recognize and process text through simple operations. You can select text in any application and trigger preset text processing actions using hotkeys, double-click, or drag-select.

## What's Inside

- **Official Website**: Landing page and project overview
- **Documentation**: Comprehensive guides and tutorials for TextGO
- **Extensions**: Collection of extensions to enhance TextGO's functionality
