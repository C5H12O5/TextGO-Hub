# Features

## Core Features

### 1. Multiple Trigger Methods

TextGO offers flexible ways to activate text processing:

- **Global Hotkeys**: Configure custom keyboard shortcuts to trigger text processing
- **Mouse Double-Click**: Double-click selected text to activate
- **Text Selection**: Automatically trigger when text is selected by dragging

Each trigger method can have independent rule configurations, allowing you to customize behavior for different scenarios.

### 2. Flexible Execution Modes

Choose between two execution modes based on your workflow:

- **Instant Execution**: Automatically execute the matched action immediately
- **Interactive Toolbar**: Show a popup toolbar with available actions for manual selection

Switch between modes freely to adapt to different use cases and preferences.

### 3. Built-in Text Recognition

TextGO comes with rich built-in text type recognition:

- URLs and web links
- Email addresses
- Phone numbers
- IP addresses
- File paths
- Dates and times
- And many more...

### 4. Custom Actions

Process recognized text with various actions:

- Open in browser
- Copy to clipboard
- Search with search engines
- Execute custom commands
- Run scripts
- And more...

### 5. Extensibility

Extend TextGO's capabilities through:

- **Regular Expressions**: Define custom patterns for text recognition
- **Machine Learning Models**: Use ML models for advanced text classification
- **Scripts**: Write custom processing logic in JavaScript or Python
- **Local AI Integration**: Connect to local AI services for intelligent text processing

### 6. Customizable Icons

Personalize your toolbar with icons:

- Choose from the built-in icon library
- Upload custom SVG icons
- Create a unique visual experience

### 7. Cross-Platform Support

Native support for both macOS and Windows with:

- Consistent user experience
- Platform-specific optimizations
- Native system integration

## Use Cases

### Web Development

- Quickly open URLs in development
- Extract and test API endpoints
- Parse JSON and XML data

### Writing & Content Creation

- Quick fact-checking with web search
- Format and transform text
- Manage references and citations

### System Administration

- Process log files
- Handle IP addresses and server information
- Execute system commands

### Daily Productivity

- Open files and folders quickly
- Manage contact information
- Process emails and phone numbers

## Configuration

All features can be configured through the intuitive settings interface:

1. Click the system tray icon
2. Select "Settings"
3. Configure triggers, actions, and rules
4. Save and start using immediately
