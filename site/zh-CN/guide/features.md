# 功能特性

## 核心特性

### 1. 多种触发方式

TextGO 提供灵活的文本处理激活方式：

- **全局快捷键**：配置自定义键盘快捷键触发文本处理
- **鼠标双击**：双击选中的文本激活
- **文本选择**：拖拽选择文本时自动触发

每种触发方式可以有独立的规则配置，允许你为不同场景自定义行为。

### 2. 灵活执行模式

根据你的工作流程选择两种执行模式：

- **立即执行**：自动立即执行匹配的动作
- **工具栏交互**：显示弹出工具栏，包含可用动作供手动选择

自由切换模式以适应不同的使用场景和个人偏好。

### 3. 内置文本识别

TextGO 内置丰富的文本类型识别：

- URL 和网页链接
- 电子邮件地址
- 电话号码
- IP 地址
- 文件路径
- 日期和时间
- 以及更多...

### 4. 自定义动作

使用各种动作处理识别的文本：

- 在浏览器中打开
- 复制到剪贴板
- 使用搜索引擎搜索
- 执行自定义命令
- 运行脚本
- 以及更多...

### 5. 高度可扩展

通过以下方式扩展 TextGO 的能力：

- **正则表达式**：定义自定义模式进行文本识别
- **机器学习模型**：使用 ML 模型进行高级文本分类
- **脚本**：用 JavaScript 或 Python 编写自定义处理逻辑
- **本地 AI 集成**：连接到本地 AI 服务进行智能文本处理

### 6. 个性化图标

使用图标个性化你的工具栏：

- 从内置图标库中选择
- 上传自定义 SVG 图标
- 创建独特的视觉体验

### 7. 跨平台支持

原生支持 macOS 和 Windows，提供：

- 一致的用户体验
- 特定平台优化
- 原生系统集成

## 使用场景

### Web 开发

- 快速打开开发中的 URL
- 提取和测试 API 端点
- 解析 JSON 和 XML 数据

### 写作与内容创作

- 快速网络搜索事实核查
- 格式化和转换文本
- 管理参考文献和引用

### 系统管理

- 处理日志文件
- 处理 IP 地址和服务器信息
- 执行系统命令

### 日常效率

- 快速打开文件和文件夹
- 管理联系信息
- 处理电子邮件和电话号码

## 配置

所有功能都可以通过直观的设置界面进行配置：

1. 点击系统托盘图标
2. 选择"设置"
3. 配置触发器、动作和规则
4. 保存并立即开始使用
