# 快速开始

本指南将帮助你快速安装和配置 TextGO，提升文本处理效率。

## 系统要求

### macOS

- macOS 11 或更高版本
- 需要授予辅助功能权限

### Windows

- Windows 10 或更高版本
- 64 位操作系统

## 下载安装

1. 访问 [GitHub Releases](https://github.com/C5H12O5/TextGO/releases) 页面
2. 下载适合你系统的安装包：
   - macOS（Apple Silicon）：`TextGO_x.x.x_aarch64.dmg`
   - macOS（Intel）：`TextGO_x.x.x_x64.dmg`
   - Windows：`TextGO_x.x.x_x64-setup.exe`

## 权限设置

### macOS 必需权限

TextGO 在 macOS 上需要以下权限才能正常工作：

#### 1. 辅助功能权限

用于模拟键盘输入，实现文本替换等功能。

**设置步骤**：

1. 打开"系统设置">"隐私与安全性">"辅助功能"
2. 找到 TextGO 并勾选
3. 如未出现，点击"+"号手动添加 TextGO

> 💡 应用首次使用这些功能时，系统会自动提示授权。

### Windows 权限

Windows 版本无需额外权限设置，安装后即可使用。

## 首次运行

1. **启动应用**
   - macOS：从 Launchpad 或 Applications 文件夹启动
   - Windows：从开始菜单或桌面快捷方式启动

2. **系统托盘**

   应用启动后会在系统托盘显示图标，点击可以：
   - 显示/隐藏主窗口
   - 打开设置页面
   - 退出应用

## 基本使用

1. 打开快捷键设置页面
2. 点击`"注册快捷键"`按钮
3. 选择`"鼠标双击选中"`或`"鼠标拖拽选中"`
4. 执行模式使用默认的`"工具栏模式"`
5. 在规则列表中点击`"+"`号新增规则
6. 选择识别类型和执行动作
7. 点击确定保存配置

现在，双击或拖拽选中文本后会出现浮动工具栏，选择预设动作即可处理文本。

## 配置说明

### 执行模式

- **静默模式**：选中文本后立即执行动作，适合简单转换操作
- **工具栏模式**：显示浮动工具栏供选择，适合需要多种动作的场景

### 输出模式

- **替换**：执行结果直接替换选中文本
- **弹窗**：在弹窗中显示执行结果，可进一步编辑或复制
