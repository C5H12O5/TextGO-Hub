# 注册快捷键

TextGO 支持三种触发方式：键盘快捷键、鼠标双击和拖拽选中。每种方式都可以独立配置规则，本文将详细介绍快捷键的注册和管理方法。

## 添加快捷键

### 步骤一：进入快捷键设置

1. 点击系统托盘图标
2. 选择"管理快捷键"
3. 点击右上角的"注册快捷键"按钮

### 步骤二：选择快捷键类型

在弹出的下拉列表中选择快捷键类型：

- **鼠标双击选中**
- **鼠标拖拽选中**
- **键盘按键触发**

### 步骤三：添加规则

在对应快捷键下方的规则列表中，点击"+"号添加新规则，配置以下内容：

#### 1. 识别类型

选择要识别的文本类型，支持以下选项：

**跳过识别**

适用于所有文本，无需进行识别

**内置类型**

- 常规：包括邮箱、URL 等常见文本类型
- 命名格式：包括驼峰、下划线等代码命名格式
- 自然语言：包括英语、中文等自然语言文本
- 编程语言：包括 JavaScript、Python 等编程语言代码片段

**自定义类型**

- 正则表达式：通过自定义正则表达式匹配特定文本
- 机器学习模型：使用预训练的机器学习模型进行文本分类

#### 2. 执行动作

选择文本识别后要执行的动作：

**内置动作**

- 复制：将文本复制到剪贴板
- 常规：包括打开链接、打开文件等常用操作
- 格式转换：实现驼峰、下划线等命名格式之间的相互转换
- 文本处理：包括去除前后空格等文本处理操作

**自定义动作**

- 搜索：通过指定搜索引擎搜索文本内容
- 脚本：执行自定义的 JavaScript 或 Python 脚本来处理文本
- AI 对话：调用本地 AI 模型处理文本内容

#### 3. 高级选项

**工具栏显示**

- 设置是否在工具栏中显示图标或文本
- 设置是否在工具栏中显示执行结果

**执行结果**

- 选择结果的输出模式
  - **替换**：使用执行结果直接替换选中的文本
  - **弹窗**：在弹窗中显示执行结果，支持进一步编辑或复制
- 设置是否将结果保存到历史记录
- 设置是否自动复制结果到剪贴板

## 管理快捷键

### 编辑规则

1. 点击规则右侧的编辑图标，可以修改规则配置
2. 选中规则后点击删除图标，可以删除该规则

### 调整顺序

选中规则后，可以通过上下箭头按钮调整规则的执行顺序。规则顺序的影响如下：

1. 静默模式下，按从上到下的顺序依次匹配规则
2. 工具栏模式下，按从上到下的顺序在工具栏中显示

## 切换执行模式

TextGO 提供两种执行模式：**静默模式**和**工具栏模式**。不同的模式适用于不同的使用场景，合理选择可以大幅提升工作效率。

### 静默模式

选中文本并触发后，**立即执行动作**，无需额外操作。

**特点**：

- 快速高效，一气呵成
- 适合确定性的处理任务
- 无界面干扰，专注当前工作

**适用场景**：

- 简单的文本转换操作（例如命名风格转换）
- 明确的动作指令（例如打开 URL、复制文本）
- 需要快速连续操作的场景

### 工具栏模式

选中文本并触发后，**显示浮动工具栏**，可以从多个动作中进行选择。

**特点**：

- 直观的可视化界面
- 支持多种处理方式
- 可以预览执行结果

**适用场景**：

- 同一文本有多种可选的处理方式
- 需要先预览处理结果再做决定
- 不确定使用哪种动作时
